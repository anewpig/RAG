# PolicyLens RAG

A citation-grounded RAG project for internal knowledge base QA.

## Goal

Build a practical RAG system that supports:

- document cleaning
- chunking
- embeddings
- vector storage
- retrieval
- generation
- citation tracing
- basic evaluation
